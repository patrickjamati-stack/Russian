⚠️ Important — how to open this file
Firebase requires this file to be served over http://, not opened directly (file://).

Easiest option — VS Code:
Install the Live Server extension → right-click the file → Open with Live Server.

Or use Python (any computer):
Open a terminal in the same folder and run:
python3 -m http.server 8080
Then open http://localhost:8080/russian_tutor.html

Also set Firestore rules:
In Firebase Console → Firestore → Rules, paste:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} {
      allow read, write: if true;
    }
  }
}
